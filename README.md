# 🌟 Thought of the Day – Quote Generator

A simple and elegant **Quote Generator Web App** built using **HTML, CSS, and JavaScript**.
It fetches random inspirational quotes from an online API and allows users to instantly share them.

---

## 🚀 Features

* 🎯 Fetches random quotes dynamically using API
* 🔄 Generate a new quote with one click
* 🐦 Share quotes directly on **X (Twitter)**
* 🎨 Clean and responsive UI design
* ⚡ Fast and lightweight (Vanilla JavaScript)

---

## 🛠️ Built With

* **HTML5** – Structure
* **CSS3** – Styling & Layout
* **JavaScript (ES6)** – Logic & API Fetching
* **DummyJSON Quotes API**
